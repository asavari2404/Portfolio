PDF AI Assistant

Codelab

Pipeline Flowchart

Users upload or select a PDF in Streamlit, FastAPI handles parsing and orchestration, LiteLLM routes requests to the chosen LLM, Redis caches extracted content, and S3 persists parsed files for reuse.

Tooling Stack Used

  • FastAPI, Streamlit, LiteLLM
  • Redis caching, AWS S3 storage
  • Docker Compose deployment

KPIs & Business Impact

  • Tokens processed per request and estimated cost per model
  • Summary and Q&A response time
  • Cache hit rate for repeated documents

Summary Insights + Decisions

  • Multi-LLM support through LiteLLM to compare model quality and cost trade-offs.
  • Redis caching to reduce repeated LLM calls and improve response latency.
  • Stored parsed files in S3 for durability and reuse across sessions.

SEC Financial ETL

Codelab

Pipeline Flowchart

Airflow schedules SEC data ingestion, raw files land in S3, data is transformed to JSON and then DBT models build structured tables in Snowflake, which are served through FastAPI and Streamlit for analyst queries.

Tooling Stack Used

  • Apache Airflow, DBT, Snowflake
  • FastAPI, Streamlit, AWS S3

KPIs & Business Impact

  • Pipeline run time per quarter and end-to-end latency
  • Data quality test pass rate (DBT)
  • Query latency on denormalized tables vs JSON

Summary Insights + Decisions

  • Kept raw staging for auditability and JSON for flexible querying.
  • Denormalized fact tables prioritized for analyst query performance.
  • Airflow orchestration + DBT validation to enforce repeatable quality checks.

NVIDIA RAG Pipeline

Codelab

Pipeline Flowchart

Airflow schedules scraping of NVIDIA quarterly PDFs, parsers extract text and tag quarter metadata, embeddings are stored in Pinecone, retrieval runs in naive, vector, and hybrid modes, and FastAPI + Streamlit serve RAG answers powered by GPT-4o.

Tooling Stack Used

  • Apache Airflow, requests, BeautifulSoup for ingestion
  • PyMuPDF + pdfminer.six parsing with quarter metadata
  • OpenAI embeddings + Pinecone vector store
  • FastAPI backend + Streamlit UI
  • Docker + docker-compose deployment

KPIs & Business Impact

  • Automated quarterly ingestion and repeatable refresh cadence
  • Hybrid retrieval filtered by year/quarter metadata for precision
  • Validated retrieval quality with contextual correctness checks

Summary Insights + Decisions

  • Multiple PDF parsers improved extraction reliability across reports.
  • Hybrid retrieval balances recall and precision for quarter-specific queries.
  • Docker-compose enabled one-command deployment and reproducibility.
